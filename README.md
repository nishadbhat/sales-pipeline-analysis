# sales-pipeline-analysis
Sales pipeline performance analysis using SQL and Tableau

Project Overview

This project analyzes sales pipeline performance to evaluate revenue trends, sales rep effectiveness, and opportunity distribution across stages and industries. The objective was to identify conversion bottlenecks, highlight top-performing representatives, and uncover revenue concentration patterns to support data-driven sales strategy decisions.

Using SQL for data extraction and Tableau for visualization, the analysis focuses on:

- Sales rep performance benchmarking through a dynamic leaderboard
- Opportunity volume distribution across pipeline stages to detect drop-offs
- Revenue contribution by industry segment to identify high-value markets
- Closing value trends over time to assess revenue momentum and seasonality
